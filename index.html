Floristgården

An advanced website created for the fictional company Floristgården. The website includes a function that allows you to check if the company delivers flowers to your zipcode and a list of closed dates, sorted in real time. To complement this the website also includes an integrated map from Google Maps and text that informs visitors if the shop is open and when it closes, also updated in real time. To top it all off, the project has automated tests using the Selenium framework and Python's unittest framework. and also automatic code validation for HTML5 and CSS3, which is run using GitLab CI/CD.

Lunch NFC system

A small project we did for our school during the Covid-19 pandemic to ensure that the students kept their assigned lunchtime, by scanning their personal NFC-tag. The features include checking if the tag belongs to a student at the school, and which class the student scanning the class belongs to. Then the program checks the schedule of said student to see if it is their lunchtime.

Twitter livefeed search
